# Premium Living Furniture - Search, Select Options, Language and Theme Version

## Added in this version
- Search box on dashboard tables and every major module page.
- Filter/select ComboBox next to search box.
- Select/option ComboBox fields in forms, e.g. Status, Priority, Warehouse, Supplier, Role, Currency.
- Global language selector in the top header: English, Traditional Chinese, Simplified Chinese.
- Global theme selector in the top header: Light, Dark, Blue.
- Controls Demo page still includes radio, text, checkbox, popup, range slider, select/options.

1. Open `PremiumLivingFurnitureWinForms.csproj` in Visual Studio 2022.
2. Press F5.
